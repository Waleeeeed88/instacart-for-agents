# Instacart API Skills

Project-local skills for agents operating this Instacart-native API. These are not global Hermes skills; they are repository guidance files that travel with the API.

- `instacart-phone-otp-login/` — Phone-number OTP login workflow for not-logged-in users.
- `instacart-grocery-planning/` — live grocery cart planning and mutation workflow.
- `instacart-api/` — API operation, endpoints, safety model, and controller setup.
- `instacart-store-discovery/` — address-aware store comparison, including Food Basics, Costco, Walmart, Adonis, and other available Instacart.ca stores.
